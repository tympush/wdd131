Warhammer 40,000 Army Builder

Site Name (Warhammer 40,000 Army Builder):

My site name is very simple, clear, and direct to the point, telling users exactly what the site is about. It's an army list builder for the game Warhammer 40,000.

Site Purpose:

The sites provides users with an army list building tool for the game, featuring several factions, as well as presenting information and statistics about those factions.

Scenarios:

  • How many points does a chaos rhino cost?
  • How many Exalted Eightbound can I have in my army?
  • What is the story behind the Imperial Knights faction?
  • Which faction has the highest win rate?

Colors:

Each faction in the game needs its own unique color, to be easily identifiable and to tell something about the faction. This means that I'm not able to select a certain color to represent the whole website, as colorful colors are associated with the factions. The game of Warhammer is very diverce and different factions invoke many different emotions, so the website will feature a veriety of faction colors. But for the main background and text, I will need generic colors.
Dark mode themes are heavily prefered among gamers, so it will be best fitting for the site's demographic. Because of this, I have chosen very dark greys for my main background colors. To contrast this, I will be using white as my font color. To make the white easily readable, the "colorful" colors will also be of darker shades.
